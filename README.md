#  Releases - Lemos

Este branch é destinado exclusivamente ao armazenamento e distribuição das versões compiladas (**APK/AAB**) do projeto.

---

## 📌 Sobre este Branch

- **Propósito:** Hospedar os arquivos de release gerados a partir do repositório privado de desenvolvimento  
- **Conteúdo:** Apenas binários e logs de versão  
- **Sincronização:** Atualizado automaticamente a cada nova milestone de desenvolvimento
